# 🐉 Pokemon World

A dynamic, interactive Pokemon encyclopedia built with modern web technologies. This application allows users to explore the vast world of Pokemon with real-time data, rich visuals, and interactive features.

## ✨ Features

- **🔍 Live Search:** Quickly find your favorite Pokemon by name.
- **🏷️ Advanced Filtering:** Filter Pokemon by **Region** (Kanto to Paldea) and **Type** (Fire, Water, Grass, etc.).
- **♾️ Infinite Scroll:** Seamlessly explore the Pokedex with automatic data loading as you scroll.
- **📱 Detailed Popups:** Click on any Pokemon to see:
    - Height, Weight, and Types.
    - Top Moves and Abilities.
    - Base Stats (HP, Attack, Defense, etc.).
    - **Evolution Chain:** Visual representation of the Pokemon's evolution line.
    - **Shiny Version:** Toggle between normal and shiny forms.
    - **Pokemon Cries:** Play the actual in-game cry of the Pokemon.
- **🎨 Dynamic Styling:** Cards are dynamically themed based on the Pokemon's primary type.
- **🌙 Dark Mode Aesthetic:** A sleek, modern dark interface for a premium look.
- **⚡ Responsive Design:** Optimized for both desktop and mobile viewing.

## 🚀 Tech Stack

- **HTML5:** Semantic structure.
- **CSS3:** Advanced styling, including Flexbox, CSS Grid, animations, and custom scrollbars.
- **JavaScript (Vanilla):** Core logic, DOM manipulation, and asynchronous API handling.
- **PokeAPI:** Powers the application with comprehensive, real-time Pokemon data.

## 🛠️ How to Run

1.  **Clone or Download:** Save the project folder to your local machine.
2.  **Open in Browser:** Simply double-click `index.html` or open it using a local server (like Live Server in VS Code) for the best experience.
3.  **Explore:** Start searching, filtering, and clicking on Pokemon to dive into their details!
